# Snip-Cache Demo Execution Guide
This project provides a demo environment for generating and evaluating code-based cache entries using the Fluent Speech Commands (FSC) dataset.  
Follow the steps below to set up the environment and run the entrypoint script (`demo.py`).

## 1. Environment Setup
### Python Version
- Python 3.10 or later recommended

### Install Dependencies
Install all required packages using:
```bash
pip install -r requirements.txt
```

## 2. Environment Variables (.env)
Create a .env file in the project root and set the following values:
```bash
OPENAI_API_KEY=YOUR_API_KEY
```

## 3. How to Run
```bash
python demo.py
```
